Courses I Have Taken

  • CS50 by Harvard University

    CS50 is a comprehensive introduction to computer science offered by Harvard University. This course covers a broad range of foundational topics including algorithms, data structures, software engineering, and web development. It's known for its rigorous approach, offering a deep dive into the theory and practice of programming. The course also emphasizes problem-solving skills, encouraging students to think algorithmically and creatively about solutions.

  • Responsive Web Design by FreeCodeCamp

    The Responsive Web Design course by FreeCodeCamp is focused on teaching the principles and practices of designing web applications that adapt to various devices and screen sizes. This course covers HTML5, CSS3, and techniques like Flexbox and CSS Grid, which are essential for modern web design. It's an excellent resource for understanding how to create user-friendly, accessible, and mobile-responsive websites.

  • JavaScript Algorithms and Data Structures by FreeCodeCamp

    This course delves into the core concepts of JavaScript, one of the most popular programming languages in the world of web development. It offers in-depth knowledge about various data structures like arrays, stacks, queues, and linked lists, and explores algorithms for manipulating these structures. This course is crucial for developing efficient code and understanding how to tackle complex problems in software development.
